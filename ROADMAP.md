# Roadmap

## Planned Features
- Additional tools in the Tools Installer (7-Zip, Notepad++, more)
- Tool install status detection and update notifications
- Tool categories and search
- UI profiles (Classic, Gaming, Minimal)
- Enhanced logging and diagnostics panel
- Optional auto-update system for tools

## Under Consideration
- Offline tool cache
- Exportable system reports
- Advanced Insider build warnings
